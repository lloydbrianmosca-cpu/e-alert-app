# e-Alert App

Emergency Response System with React Native and Expo.

## Getting Started

### Prerequisites
- Node.js installed
- Expo Go app installed on your physical device (v54 or later)

### Installation

1. Navigate to the project directory
2. Install dependencies:
   ```bash
   npm install
   ```

### Running on Physical Device

1. Start the Expo server:
   ```bash
   npm start
   ```

2. Scan the QR code with your physical device using Expo Go v54
3. The app will load on your device

## Project Structure

- `App.js` - Main sign-in screen component
- `index.js` - Entry point
- `app.json` - Expo configuration
- `package.json` - Dependencies and scripts

## Features

- Emergency-themed sign-in UI (Red & White)
- Email and password input fields
- Password visibility toggle
- "Forgot Password" link
- Emergency notice banner
- Clean, professional design optimized for mobile
- No gradients (flat design)

## Colors

- Primary Red: #DC2626 (Emergency alert color)
- White: #FFFFFF
- Dark Gray: #1F2937
- Light Gray: #F9FAFB, #E5E7EB
